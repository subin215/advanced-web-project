About the project:
Why?
This project was created for the purpose of CS-389: Advanced Web and Cloud Development class being taught in Carroll College.
What does it do?
A signed in user can use this application to convert between three currencies. This was a front-end rule assigned by us. In reality, the API can handle any currency exchange as long as the proper currency identifier is used.
What's cool about it?
Everything and nothing? We focused on building a simple application with a robust and unit tested service layer. We connected our repository to Travis-CI so that it would build and test for every push to the repository. Our CI connected to our GitHub repository for the project and builds every time we push to GitHub. We also wrote a script to deploy our application as a docker container to our EC2 instance in AWS. We use docker-compose to build the application container and deploy it to docker in EC2 instance. The application connects to AWS RDS instance for database.
