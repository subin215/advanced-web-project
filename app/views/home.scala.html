Advanced Web and Cloud Development

This elective class is intended to give the student an introduction to real-world development using a modern web-application framework as used by commercial development staff. The student will use Java to create an application to perform a task of their choice, all the while learning about web frameworks, HTML, dependency-injection, and unit testing. The course will cover a wide variety of topics including software development, configuration, deployment, and monitoring. Specific technologies and frameworks to be covered include, Java 8, Scala 2.11, HTML5, Play Framework, Spring, Docker, JUnit, MySQL(SQL), Redis (NoSQL), and web-services (RestFUL). Time permitting, the students will upload their completed projects to an AWS instance in the Amazon cloud and verify that it can be accessed. -Class Rationale, Professor Nate Williams

Professor Williams is well educated and has covered several topics that are key to our success as web and cloud developers.
